트리 순회(Tree Traversal)

Preorder 순회

루트(Root)를 먼저 방문한 후 왼쪽 서브트리 → 오른쪽 서브트리 순서로 탐색한다.

A → B → D → E → C → F → G

Inorder 순회

왼쪽 서브트리를 먼저 방문한 후 루트(Root)를 방문하고 마지막으로 오른쪽 서브트리를 탐색한다.

D → B → E → A → F → C → G

Postorder 순회

왼쪽 서브트리 → 오른쪽 서브트리를 먼저 방문한 뒤 마지막에 루트(Root)를 방문한다.

D → E → B → F → G → C → A
