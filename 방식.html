필기시험 방식

정보처리기능사 필기시험은 종이 시험지가 아닌 컴퓨터 화면을 이용하여 진행됩니다. 응시자는 모니터에 표시된 문제를 읽고 마우스로 정답을 선택하며, 시험 종료 후 즉시 점수와 합격 여부를 확인할 수 있습니다.

🖥️

컴퓨터 기반 시험(CBT)

종이 시험지 대신 모니터를 보고 마우스로 답안을 클릭하는 방식입니다. *참고: 신분증 필참

✔️

객관식 4지 선다형

모든 문제는 4개의 보기 중 정답 1개를 선택합니다.

📚

총 60문항 출제

시험 시간은 총 60분이며 60문항이 출제됩니다.

⏱️

60분 시험

평균적으로 1분당 1문제를 해결하는 속도로 시험이 진행됩니다.

📊

즉시 결과 확인

시험 종료 버튼을 누르면 점수와 합격 여부를 바로 확인할 수 있습니다.

🏆

합격 기준

100점 만점 기준 60점 이상(36문제 이상 정답) 획득 시 합격입니다.

실기시험 방식

정보처리기능사 실기시험은 컴퓨터로 직접 프로그램을 실행하는 방식이 아닌 지필형(필답형) 시험으로 진행됩니다. 응시자는 문제를 읽고 답안을 직접 작성해야 하며, 프로그래밍 언어와 SQL 관련 문제의 결과를 분석하거나 빈칸을 채우는 형태로 출제됩니다.

✍️

지필형 시험(필답형)

컴퓨터로 직접 코딩을 실행하는 방식이 아니라 시험지에 직접 답을 손으로 적는 방식입니다. *참고: 신분증 필참

📝

주관식 문항

객관식 보기 없이 단답형 또는 서술형 문제로 출제됩니다.

📚

약 12문항 출제

시험 시간은 총 90분이며 약 12문항 내외가 출제됩니다.

⏰

90분 시험

문제를 분석하고 코드를 작성해야 하므로 충분한 풀이 시간이 제공됩니다.

💻

프로그래밍 언어 포함

SQL, Java, Python 등의 코드 실행 결과를 예측하거나 빈칸에 들어갈 코드를 작성해야 합니다.

🏆

합격 기준

필기와 동일하게 100점 만점 중 60점 이상을 받으면최종 합격입니다.
