Library

Library

  1. 표준 라이브러리 : 프로그래밍 언어에 기본적으로 포함되어 있는 라이브러리로, 여러 종류의 모듈이나 패키지 형태이다.
  2. 외부 라이브러리 : 개발자들이 필요한 기능들을 만들어 인터넷 등에 공유해 놓은 것으로, 다운을 받아 설치한 후 사용한다.
  3. 모듈 : 하나의 기능이 한 개의 파일에 구현된 형태
  4. 패키지 : 하나의 패키지 폴더 안에 여러 개의 모듈을 모아 놓은 형태
A.compareTo(B)
뜻

숫자로 된 문자열 A와 B를 비교하여 같으면 0, A가 크면 1, B가 크면 -1을 반환

A.equals(B)
뜻

대소문자를 구분하여 문자열 A와 B를 비교한 후 같으면 참, 다르면 거짓을 반환함

A.equalsIgnoreCase(B)
뜻

대소문자를 구분하지 않고 문자열 A와 B를 비교한 후 같으면 참, 다르면 거짓을 반환함

toLowerCase(문자열)
뜻

문자열을 모두 소문자로 변환함

toUpperCase(문자열)
뜻

문자열을 모두 대문자로 변환함

split(구분자)
뜻

지정된 구분자로 문자열을 구분하여 분리함

A.length()
뜻

문자열 A의 길이를 반환함

A.trim()
뜻

문자열 A의 좌우 공백을 제거함
