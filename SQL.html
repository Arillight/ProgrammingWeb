SQL

DDL

명령어 기능
CREATE 데이터베이스 객체를 생성한다.
ALTER 데이터베이스 객체를 수정한다.
DROP 데이터베이스 객체를 삭제한다.

DML

명령어 기능
SELECT 테이블에 조건에 맞는 튜플을 검색한다.
INSERT 테이블에 새로운 튜플을 삽입한다.
DELETE 테이블에 조건에 맞는 튜플을 삭제한다.
UPDATE 테이블에 조건에 맞는 튜플의 내용을 수정한다.

DCL

명령어 기능
COMMIT 작업 결과를 실제 데이터베이스에 반영한다.
ROLLBACK 작업을 이전 상태로 되돌린다.
GRANT 사용 권한을 부여한다.
REVOKE 사용 권한을 취소한다.
