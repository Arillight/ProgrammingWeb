Linux

명령어 기능
cd 디렉터리의 위치를 변경한다.
chmod 파일의 보호 모드를 설정하여 파일의 사용허가를 지정한다.
chown 파일 소유자와 그룹을 변경한다.
cat 파일 내용을 화면에 표시한다.
명령어 기능
ls 현재 디렉터리의 파일 목록을 표시한다.
mkdir 디렉터리를 생성한다.
rmdir 디렉터리를 삭제한다.
pwd 현재 작업중인 디렉터리 경로를 화면에 표시한다.
tar 파일을 압축하거나 압축을 해제한다.
